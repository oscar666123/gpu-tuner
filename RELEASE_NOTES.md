# GPU Tuner v0.1.0

Initial release.

## Highlights

- NVIDIA GPU telemetry dashboard
- Power limit tuning
- Locked core clock and locked memory clock
- Core and memory clock offset paths through NVML with fallback support
- Per-setting Apply checkboxes
- Local profiles, safety threshold, and logs
- Windows NSIS and MSI installers

## Notes

- Run as administrator.
- NVIDIA driver support varies by GPU and driver branch.
- Voltage and fan controls are intentionally out of scope.
